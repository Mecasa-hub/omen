├── Branded: OMEN logo, purple/gold gradient
├── Includes: user's referral code
└── One-click share to X/Twitter
        │
        ▼
Followers see the card on their timeline
├── "How did you know that?!"
├── "What is OMEN?"
└── Click referral link → Sign up
        │
        ▼
New user signs up with referral code
├── Original user gets 10% bonus credits
└── New user starts making predictions

        ▼
♻️ New user wins → Generates their own brag card → Loop repeats

| Raw Polymarket | No AI analysis | 5-agent debate engine with confidence scores |

### Positioning Statement

> **OMEN is the Bloomberg Terminal for Polymarket** — combining AI-powered predictions, whale intelligence, and one-click trading in a platform designed to go viral.

### Key Differentiators

1. **AI Debate Engine** — Not just one AI model, but 5 specialized agents debating each other. Users watch the debate unfold in real-time via WebSocket.

2. **Whale Intelligence** — The only platform that tracks top Polymarket wallets AND lets you copy-trade them with configurable risk settings.

3. **Social-First Design** — Every feature designed with sharing in mind. Brag cards, whale alerts, referral codes embedded throughout.

4. **Pay-as-you-go** — No $99/month subscriptions. $5 gets you 50 predictions. Lowest barrier to entry in the market.
