## вңЁ Features

### рҹ”® Oracle Chamber вҖ” Real AI Debates

The Oracle uses a **5-agent swarm** powered by **Gemini 2.0 Flash** with distinct personalities:

| Agent | Role | Style |
|-------|------|-------|
| рҹҹў **Atlas** | Bull Analyst | Finds reasons to buy |
| рҹ”ҙ **Nemesis** | Bear Analyst | Finds reasons to sell |
| рҹ”ө **Quant** | Statistician | Pure data and probability |
| рҹҹЎ **Maverick** | Contrarian | Challenges consensus |
| рҹҹЈ **Clio** | Historian | Historical patterns and precedent |

Agents make **real LLM API calls** вҶ’ Debate independently вҶ’ Vote вҶ’ 1,200-dot swarm matrix animates вҶ’ Verdict with confidence score.

### вҡ”пёҸ War Room вҖ” 1,200 AI Agents Vote Live

- Each agent card reveals sequentially with real AI reasoning
- **1,200-dot swarm matrix** with ripple animation shows consensus forming
- Green = YES, Red = NO вҖ” watch the swarm converge in real-time
- Whale agreement dots overlay for confidence boost
- Final verdict card with confidence percentage

### рҹҗӢ Whale Intelligence вҖ” Live On-Chain

- **Live Polygon blockchain tracking** via `polygon-bor-rpc.publicnode.com`
- Real MATIC and USDC balances for tracked wallets
- Transaction count and recent volume monitoring
- **Polymarket CTF Exchange detection** вҖ” flags trades to Polymarket contracts
- DB whales with historical stats: win rate, PnL, volume, specialty
- **Copy any whale** вҖ” one click to mirror their latest trade

### рҹ“Ҡ Trading вҖ” Three Modes

| Mode | Description |
|------|-------------|
| **Manual** | Browse live Polymarket markets вҶ’ select вҶ’ set price/size вҶ’ place limit order |
| **Oracle Auto-Trade** | Ask Oracle вҶ’ AI analyzes вҶ’ auto-places bet on matching Polymarket market |
| **Copy Trade** | Enter whale address вҶ’ mirrors their latest Polymarket position |

**Safety built-in:**
- рҹ”’ **API-only credentials** вҖ” no private keys stored, only Polymarket API creds
- рҹ“Ҡ **Liquidity check** вҖ” rejects trades on illiquid orderbooks (spread > $0.50)
- рҹӣЎпёҸ **Risk controls** вҖ” max bet $50, daily limit $200, stop-loss 35%
- рҹ”җ **Fernet encryption** вҖ” all credentials encrypted at rest

### рҹ’° Crypto Payments вҖ” Polygon

- **Wallet:** `0x135C480C813451eF443A2F60cfaD49EA7197B855`
- **Accepted:** MATIC, USDC, USDC.e on Polygon
- **Verification:** On-chain transaction hash verification via Polygon RPC

## рҹҡҖ Quick Start

### Quick Deploy (Recommended)

```bash
git clone https://github.com/Mecasa-hub/omen.git
cd omen
cp .env.example .env  # Add your OpenRouter API key
pip install aiosqlite httpx aiohttp pydantic uvicorn fastapi cryptography py-clob-client
python -m uvicorn deploy:app --host 0.0.0.0 --port 8888
# OMEN is live at http://localhost:8888
```

### Docker

```bash
git clone https://github.com/Mecasa-hub/omen.git
cd omen
cp .env.example .env
docker-compose up -d
```

### Environment Variables

```env

# Polymarket (users provide their own API keys in the UI)
# No server-side trading keys needed
```

- **45 Named AI Agents** вҖ” Each with unique personality, backstory, cognitive biases, and expertise

- **Multi-Round Debate System:**
  - **Round 1**: All 45 agents vote independently with research context
  - **Round 2**: Top 15 most confident agents re-evaluate after seeing Round 1 results
  - **Faction Formation**: Agents cluster into Strong Bulls, Cautious Bulls, Strong Bears, Cautious Bears
  - **Vote Flips**: Track which agents changed their minds during debate and why
- **Real-Time Event Stream** вҖ” Scrolling terminal log of agent activities
- **Performance Metrics** вҖ” Throughput, latency, OPS/s, memory, CPU, queue, errors, tokens
- **D3.js Force Graph** вҖ” 45 agent nodes with connections and operation labels
- **Boot Animation** вҖ” Agents come online 0/45 вҶ’ 45/45 with status messages
- **Obsidian-Style Memory Vault** вҖ” Each agent has persistent memory stored in markdown files
- **Autoresearch-Inspired Learning** вҖ” Agents track hypotheses, evidence, and post-mortems
